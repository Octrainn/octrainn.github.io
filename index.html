Fault Finder

I participated in a hackathon project alongside two undergraduate partners. Together, we successfully completed the project within a two-day timeframe. Our objective was to employ OpenAI's API to generate interview-level questions paired with deliberately incorrect solutions. I took charge of implementing the project's core functionality by creating a backend database using Node JS. The system we developed selects a question for the user and presents them with an intentionally incorrect solution. The user's task is to review and debug the provided code. Once the debugging process is finished, the system provides feedback on the accuracy of the debugging. Check out the project!
